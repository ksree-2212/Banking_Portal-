# 🏦 BankingPortal-API

**BankingPortal-API** is a secure Java Spring Boot backend for digital banking applications.  
It provides RESTful APIs for user registration, authentication, deposits, withdrawals, and fund transfers — built with scalability, security, and data integrity in mind.

---

## 🚀 Features

- **User Management:** Registration & JWT-based authentication  
- **Account Operations:** Deposit, withdrawal, and balance check  
- **Fund Transfer:** Secure inter-account transactions  
- **Security:** Spring Security + JWT for stateless protection  
- **Database:** MySQL integration via Spring Data JPA (Hibernate)  
- **Error Handling:** Centralized global exception management  

---

## 🏗️ Architecture

The system follows a **3-tier architecture**:
1. **Controller Layer:** Handles HTTP requests and responses.  
2. **Service Layer:** Contains business logic and transaction handling.  
3. **Repository Layer:** Performs CRUD operations using JPA.
